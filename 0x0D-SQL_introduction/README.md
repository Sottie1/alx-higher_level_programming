# 0x0D. SQL - Introduction
<h2>Install MySQL 8.0 on Ubuntu 20.04 LTS</h2>
<p>$ sudo apt update</p>
<p>$ sudo apt install mysql-server</p>
<p>$ mysql --version
mysql  Ver 8.0.25-0ubuntu0.20.04.1 for Linux on x86_64 ((Ubuntu))
$</p>
<h3>Connect to your MySQL server:</h3>
<p>$ sudo mysql</p>
